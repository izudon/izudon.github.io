# 昨日の回線故障

* まず、偶然故障したのか、故意によるものかということがある。
* 故意によるものなら、誰が、何のために、どうやって、  
  故障させたのかということになる。
* 偶然と考えるにはあまりにもタイミングが良かったものだから、  
  何者かによる故意ではないかと考えてしまうのだけれど、  
  誰が、何のために、どうやってというところは、わからない。
