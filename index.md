### 好きな領域の自覚

* 「好き」にはおそらく二通りの意味があって・・・。
* 「好き」を基準として持ち出すと、間違いや勘違いを生み出す恐れが  
  あるのではないかと僕は思っているのだが。

{% include youtube.md v="f9n9oTY-q2E" title="" %}

* 友だちとおしゃべりするのが好きです  
  （ある映画公開イベントでのファンの方（女子）の発言）
* パチンコが好き
* 競馬が好き

同じ「好き」でもこの「好き」とその「好き」は違う気がする。

どう違うのか。


* この「好き」は
  多分「[pastime](https://ejje.weblio.jp/content/pastime)」である。  
  （ごめんなさい英語持ち出さないと説明できない）
* 日本語では「娯楽」「余暇を楽しく過ごす」「気晴らし」と訳される。
* 見ての通り「通りすぎる（pass）時間（time）」という語源を持つ。

おしゃべりもパチンコも競馬も確かに「楽しい」けど・・・。

* 「物足りない」ものでもあると思う。
  * 困難に立ち向かっている感覚には乏しいと思う。
  * だからといって、困難に立ち向かえば良いというものでもなく、  
    立ち向かわなくても良い困難には立ち向かわなくて良いのであるが。

その「好き」は、たぶんまた違ったニュアンスの「好き」ではないか。

・・・という長い注釈を経た上で。

* 「好き」を入り口にして考えていくという視点は新鮮味があった。
* 「好き」の２通りの意味を弁別するだけで苦労していたくらいなので、  
  自分は到底そんな発想になど思い至れているわけもなかったので。
* 本書ではこれを「自覚」することも必要だと言っている。

## 成功と失敗

{% include amazon.md asin="B07J4F2LRR" title="情報楽園会社" %}
{% include amazon.md asin="4198604487" title="情報楽園会社初版" %}

1. 社会人として（アパレル会社鈴屋の従業員として）店舗づくりをされた  
   経験が活きた。
1. 本格的に始めるとなったときに脱サラして（会社を辞めて）、  
   駅の反対側にも１店舗設けて完全に需要を制覇した。  
   * 枚方駅を降りる人の流れを意識し、  
     「**抑えるものは抑えた**」。
   * 僕が学生時代を過ごしたまちでよく行っていた TSUTAYA のそばに、  
     別系列の（おそらく個人経営の）レンタル屋ができたことがある。  
     TSUTAYA はすぐさま値引きキャンペーンを打って干しにかかった。
     * えらいことするなぁとその時は思ったけどそれが現実・・・。
     * この現実認識の源流は創業当初からあったのだなとわかった。
1. 増田氏も常に順風満帆だったわけではなく、失敗と成功を繰り返している。
   * ここは規模拡大のタイミングだと３００人の新卒を採ったら、  
     変なのも入ってきたりだとか。
   * 店舗立地でしくじったりだとか。
   * 特にディレクＴＶでの失敗はこたえた。
     * 海外勢に株の過半を所有され、拒否権に悩まされた。
       1. 日本社会や日本人の嗜好に関する理解がなく話が通じなかった。  
	      （Ｆ１放映権の件）  
	      －－のちに「僕に英語ができていればまた違った」と社長語る。
	   2. 放送となると免許事業となり役人とのやりとりが発生した。
	      * 役人はだいたいにおいて曖昧な返事で先延ばしにする。
		  * 事業はしかしながらタイミングである。  
		    （特に増田社長はタイミングを味方につけてこられた）
		  * 役人を動かすには「政治家」だと知ったそうだが・・・  
		    僕にはそれは諸刃の剣にまだ思える。
       3. やり方が否定されるとうまくはいかなくなる。
	      * 社員（仲間）を大事にする。
		    * 辞められるのが一番つらいそう。
			* 出資した三木谷浩史さんの楽天が成長し、  
			  含み益を伴って株式が返ってくるのはのちの話。
		  * 熱意でもって周囲に影響を与え物事を変えていく。  
		    * 絶対的権限を持った相手に取り入り、  
			  どうにか態度を軟化させるというものではなかった。
		  * 自分もそういうタイプだから（ここだけが似ている）、  
		    すごくわかるなーと思って読んでしまった。
   * 50-50 で事業を持ってはいけない。  
	 最後は誰かが責任を取り腹を括れるようでなければならないという、  
	 （京阪の偉いさんに見出され工事請負で財を成した）祖父の教えを、  
	 違えたことが結果的に悪く出てこうなったことが口惜しいと。
   * 経営者としてはしかし、リスクの存在に気付いてはいても、  
     ここがタイミングと思えば打って出なければいけない局面もあるから。
	 * 間違ったことではないと思う。
	 * 見積もった上で取れるリスクと判断して向かわれたわけですから。

あと、読んで自分まだまだだな（スキル的に足りてないな）と思ったのは、
増田さんが持たれている事業者としての金銭感覚やタイミング知覚の部分。

* 家賃相場、従業員数から人件費、  
  客層、客単価と席数、回転率から日販いくらと。  
  瞬時に見積もれるだけの感覚は僕にはまだないなと思う。

# 明朝体何使うか検討した

明朝体は Noto Serif CJK -> 游明朝体 -> Serif がいいかなと思う。

* Noto Serif CJK
  * **Good!** - フリーフォントだがそこそこ映える
	* ウェブフォントの有料フォントはまだ金銭的にハードルが高い。
	* [さくらで追加料金なしで使えるようになった](https://www.sakura.ne.jp/function/webfont/)ものの、  
	  レンタルサーバのみで、さくらVPSではまだ無理なので、  
	  レンタルサーバだと FreeBSD 固定になり制約になるので、  
	  やはりまだちょっと使えないかなという判断。
  * **Bad!** - ネットワークつながってないと見られない！
    * ダウンロードまで間を持たせるフォントを考えておかねば！
* 游明朝体
   * **Good!** - Mac と Windows 8.1 以降に標準搭載
   * **Bad!** - Android に非搭載！
* Serif - 両方ともない環境の人向けに拾う


参考：[Webにおいて明朝体はNoto Serif CJKを使うべし｜BPS株式会社](https://techracho.bpsinc.jp/nishi/2019_08_26/79504)
